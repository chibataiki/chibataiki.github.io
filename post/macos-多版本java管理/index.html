    MacOS 多版本java管理

    Posted at — Feb 28, 2020

    之前安装 Burpsuite的时候,需要的版本是JDK1.8,这次ghida则需要新版本

    linux 可以使用`sudo update-alternatives --config java` 进行java版本的选择

    macOS下则使用homebrew

    brew tap homebrew/cask-versions
brew install jenv

    显示java 环境

    /usr/libexec/java_home -v

    结果类似于

    13.0.2, x86_64:	"OpenJDK 13.0.2"	/Library/Java/JavaVirtualMachines/openjdk-13.0.2.jdk/Contents/Home
1.8.0_231, x86_64:	"Java SE 8"	/Library/Java/JavaVirtualMachines/jdk1.8.0_231.jdk/Contents/Home

    添加 jdk ,路径就是上面的结果

    jenv add <jdk_path>

    显示java版本信息:

    jenv versions
jenv -version

    设置java环境:

    jenv global [jdk version]
jenv local [jdk version]

    参考: https://gist.github.com/ntamvl/5f4dbaa8f68e6897b99682a395a44c2e
