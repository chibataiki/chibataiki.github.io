    Vim-Markdown-插件安装

    Posted at — Feb 25, 2020

    环境： macOS catalina vim: macvim

    设置vim初始化环境

    mkdir ~/.vim
mv .vimrc ~/.vim/vimrc

    vim的初始化环境在macOS上最初是～/.vimrc, 因为需要安装插件新建了.vim文件夹，所以想将.vimrc文件放入.vim目录下,这个时候需要重命名.vimrc为vimrc. 可以在:h viminfo 中查看

    安装插件

    这里使用的是vim-plug :https://github.com/junegunn/vim-plug

    需要将

    " Plugins will be downloaded under the specified directory.
call plug#begin('~/.vim/plugged')


" List ends here. Plugins become visible to Vim after this call.
call plug#end()

    添加到vimrc中,然后vim下执行:PlugInstall 就可以安装配置中的插件

    安装markdown预览插件

    https://github.com/iamcco/markdown-preview.nvim

    在vimrc Plug区域添加

    call plug#begin('~/.vim/plugged')

    预览:

    :MarkdownPreview
:MarkdownPreviewStop
