GDB 使用note

Posted at — Mar 1, 2020

gdb [program] [arguments]

反汇编 main函数
(gdb) disasseble main

修改汇编模式,可以自己选择AT&T或者Intel, 也可以在.gdbinit中修改
(gdb) set disassembly-flavor intel
(gdb) set disassembly-flavor at&t (gdb) show disassembly-flavor

At&t : mov 0xc(%ebp),%eax Intel : mov eax, DWORD PTR [ebp+0xc]

设置断点

(gdb) break *main (gdb) b *0x000055555555468a

运行
(gdb) run

继续运行
(gdb) continue / c

运行下条指令(如果调用函数,则进入函数)
(gdb) stepi / si

执行下一条指令(如果调用函数,执行完跳出)
(gdb) nexti /ni

显示寄存器信息
(gdb) info registers /i r

显示函数信息
(gdb) info functions

(gdb) set $rax=0x01 // 修改寄存器参数

显示函数:

info functions
