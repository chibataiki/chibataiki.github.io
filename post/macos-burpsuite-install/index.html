MacOS Catalina BurpSuite 安装

Posted at — Feb 24, 2020

JDK安装

burpsuite是java程序，所以需要安装JDK

JDK分为Oracle和OpenJDK，不做赘述，选择OpenJDK

下载安装比较麻烦，所以选择homebrew安装

brew cask install java 安装最新java会导致burpsuite安装失败

brew cask install java11

burpsuite 安装

下载burp-loader-keygen.jar 和 burpsuite_pro{version}.jar.

如果看到出现: -Xbootclasspath/p 说明java版本太新,java8之后这个参数就移除了.

解决方法一是使用旧版本JDK,二是自行对keygen进行patch.

安装完java8 之后, 发现mac的keygen的license在每次切换窗口后都会变化,所以我就在虚拟机中运行了keygen程序,成功激活.

证书

macOS下chrome安装:在safari下下载证书(使用burp监听端口), keychain Access 打开证书并手动信任
