macOS下IDA插件安装

Posted at — Mar 1, 2020
  • IDAFuzzy : https://github.com/Ga-ryo/IDAFuzzy 在IDA中,对函数进行模糊搜索不是由友好,所以搜了一下,发现了IDAFuzzy

IDA pro 在macOS下安装的坑

和Windows中直接将插件拖到plugin目录不同, macOS的插件安装需要两步

  1. 将ida_fuzzy.py 文件放到 plugins 目录下
  2. 将ida_fuzzy.py 所依赖的包放到 python 目录下

可以看到/Applications/IDA Pro 7.0/ida.app/Contents/MacOS中的目录结构:
