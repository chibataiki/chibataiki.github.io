sslstrip is a MITM tool.

how it works#

People may not type https:// (http://) in the address bar.

offen encouter SSL with this two types:

  1. 301/302 redirection
  2. click links with https://

sslstrip can watch http traffic and change all https links to http

arpspoof#

send ARP messages on LAN associate the attacker’s Mac address with the gateway ip, then all traffic will send too attacker’s machine

sudo arpspoof -i [interface] -t [target mac] -r [gataway mac]

traffic redirect#

sudo iptables -t nat -A PREROUTING -p tcp --destination-port 80 -j REDIRECT --to-port 10000
