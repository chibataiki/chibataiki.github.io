Simple_key_validation

March 14, 2020 · 1 min · chiba

一种简单的密码加密机制:

#include<stdio.h>
#include<stdlib.h>
int main(int argc, char *argv[]){
        if(argc==2){
                printf("Checking License: %s\n",argv[1]);
                int sum=0;
                for(int i=0;i<strlen(argv[1]);i++){
                        sum += (int)argv[1][i];
                }
//              printf("%d\n",sum);
                if( 1022 == sum){
                        printf("Accessed!");
                } else {
                        printf("Wrong!");
                }
        }else {
                printf("Usage: %s <key>\n",argv[0]);
                }
        return 0;
}

其中1022 是 “THIS IS THE KEY” 进行简单的ascii值求和的结果.

可以通过静态分析和逆向工程可以分析出加密机制.

根据逆向分析的算法生成注册码

import random

def check_key(key):
        char_sum=0
        for c in key:
                char_sum += ord(c)
        return char_sum

chars="abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789"

key = ""

while True:
        key += random.choice(chars)

        sum_temp = check_key(key)
        if sum_temp > 1022:
                key = ""
        elif sum_temp <1022:
                pass
        else:   
                print("Key:{}".format(key))
